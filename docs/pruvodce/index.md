# Průvodce

Průvodce vás provede celým procesem práce s pluginem Wall Reinforcement Wizard — od instalace přes nastavení parametrů až po generování a validaci výztuže.

## Začínáme

V sekci **Začínáme** najdete informace o požadavcích, instalaci, připojení k modelu a základním workflow.

## Průvodce funkcemi

Jednotlivé stránky popisují práci s hlavními parametry, otvory, třmínky, diagonálními pruty, preview panelem, validačním oknem a PDF exportem.
