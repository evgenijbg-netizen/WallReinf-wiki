```
# Trminky

!!! info "Verze pluginu: [VERZE]"

[One sentence: page covers stirrups, flags, and segment behavior]

## Co jsou trminky a prapory
```
Brief explanation:
- Trminky (stirrups) = closed loops connecting S1 and S2 rebar layers
- Prapory (flags) = L-shaped alternatives to stirrups for shorter wall segments
- Plugin automatically decides stirrups vs. flags based on segment size and thresholds

```
## Prahy v Nastaveni
```
Per CONTEXT.md, Settings **Trminky a prapory** section goes here:
- Two threshold sliders:
  - Stirrup threshold (default ~300mm from RebarParameters.cs StirrupThreshold) — segments shorter than this get stirrups
  - Flag threshold (default ~1500mm from RebarParameters.cs FlagThreshold) — segments shorter than this but longer than stirrup threshold get flags

!!! note "Poznamka — Logika prahu"
    Segment < prah trminku → generuji se trminky. Segment > prah trminku ale < prah praporu → generuji se prapory. Segment > prah praporu → normalni prima vyztuz bez trminku/praporu.

Screenshot placeholder: Settings window Trminky a prapory section.

```

```
- Diagonals are placed at all 4 corners of each detected opening
- 45° angle relative to opening edges
- Length determined by opening size and rebar grid
- Generated on both S1 and S2 layers
